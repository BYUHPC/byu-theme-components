Welcome to the new templates!

Feel free to check back here for updates. I'll upload the latest code every time we get a pull request on the github repository (https://github.com/byuweb/2017-components).

As of right now, you need to include the following to use the components:

  • The web component polyfills: <script src="https://lscdn.byu.edu/Template/bower_components/webcomponentsjs/webcomponents.js"></script>
  • The script file for each component you want: <script src="https://lscdn.byu.edu/Template/components/byu-[component-name]/byu-[component-name].js"></script>
  • The font-awesome stylesheet: <link rel="stylesheet" rel="stylesheet" href="https://lscdn.byu.edu/Template/bower_components/font-awesome/css/font-awesome.min.css">
  • The default site.css: <link rel="stylesheet" rel="stylesheet" href="https://lscdn.byu.edu/Template/css/site.css">
